# US-02 — Registro de usuario

## Descripción
Como usuario nuevo, quiero registrarme con mi email y contraseña
para crear una cuenta.

## Criterios de aceptación

- [ ] El usuario puede registrarse con email válido y contraseña
- [ ] El sistema valida que el email tenga formato correcto
- [ ] El sistema rechaza el registro si el email ya existe
- [ ] La contraseña debe tener mínimo 8 caracteres
- [ ] El sistema envía email de confirmación al registrarse
