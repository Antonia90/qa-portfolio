# US - 03 - Recuperación de contraseña

## Descripción
Como usuario, quiero recuperar mi contraseña
para volver a acceder a mi cuenta.

## Criterios de aceptación

- [] El usuario puede solicitar recuperación ingresando su email
- [] El sistema envía un mail con enlace de recuperación
- [] El sistema muestra error si el email no está registrado
- [] El enlace de recuperación expira pasadas las 24 horas
- [] El uruario puede establecer una nueva contraseña con el enlace
