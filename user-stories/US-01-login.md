# US-01 -- Inicio de sesión

## Descripción
Como usuario registrado, quiero iniciar sesión con mi email y contraseña para acceder a mi cuenta.

## Criterios de aceptación

- [] El usuario puede iniciar sesión con email y contraseña válidos.
- [] El sistema muestra error si el mail no existe
- [] El sistema muestra error si la contraseña es incorrecta
- [] El sistema muestra error si algún campo está vacío
- [] El sistema bloquea el acceso tras tres intentos fallidos
